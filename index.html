What it does

AI coding agents start every session from scratch. Memory Enhancer Pro adds a structured local memory store that your agent reads on startup and writes to as you work. Context, corrections, and learned preferences persist automatically between sessions.

Installation takes under 30 seconds. No server, no account, no telemetry. Memory files are plain markdown you can read, edit, or delete at any time.

How it works

  1. Local memory store created

    Creates ~/self-improving/ with three tiers: hot (last 7 days), warm (last 30 days), and cold (permanent). An optional maintenance step promotes files between tiers as they age.

  2. Agent integration configured

    The installer detects which AI agents you have installed and adds the memory store to each one's context loading path using each platform's native configuration format.

  3. Context loads automatically

    On the next session start, your agent picks up where you left off. Corrections from past sessions are applied automatically so you never repeat yourself.

  4. Restart your agent

    That's it. Your agent will have persistent memory from the next session forward. No configuration needed beyond the initial install.
